# Музыкальный стриминговый сервис
Используемые сущности: Музыкант, Альбом, Трек, Пользователь, Плейлист

Музыкант имеет атрибуты: имя, жанр

Альбом: название, год выхода, музыкант

Трек: название, музыкант

Пользователь: имя

Плейлист: название, пользователь

У музыканта будут альбомы и треки без альбомов, у пользователя - плейлисты

Бизнес-операции:
Добавление трека и привязка к альбому
Удаление всех треков в альбоме при удалении альбома
Удаление всех треков и альбомов музыканта при удалении музыканта
Удаление всех плейлистов пользователя при удалении пользователя
Объединение двух плейлистов пользователя в один
